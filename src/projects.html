Research Projects

Texera

Texera is a system to support big data analytics as a web-based service by using declarative GUI-based interactive workflows. As an undergraduate research assistant, I helped design the user interface for Texera, and implemented various functionalities (i.e. delete operators, show process information) using the Angular framework.

GitHub link: github.com/Texera/texera

Personal Projects

Fabflix

Fabflix is a movie shopping website that supports user authentication, fuzzy search, pagination, autocomplete, password encryption, and many more functionalities. The website uses MySQL for the database, Tomcat for the main server, Apache 2 for the load balancer, and AWS to host the server. I also created an Android app that implements the website’s basic functionalities.

GitHub link: github.com/Mescetina/Fabflix

Sleep Tracker

Sleep tracker is a hybrid mobile app that tracks overnight sleep time and level of sleepiness during the day. The app presents logged data in a calendar view. It sends out notifications every day to remind users to log their level of sleepiness. I used the Ionic framework to build the app, Apache Cordova to schedule daily notifications, and Ionic Storage to store user data.

GitHub link: github.com/Mescetina/sleep-tracker

UCI Search Engine

UCI search engine is a campus-wide search engine that supports natural language processing. Given a search query, the program produces a list of websites within the uci.edu domain in decreasing order of relevance. This project is a group effort. As a group member, I implemented inverted index for full text search, and tf-idf for result prioritization.

GitHub link: github.com/Mescetina/search-engine

License Plate Recognition System

License plate recognition system is a program that recognizes characters on license plates from raw images. The system is realized using convolutional neural network. It first processes the raw images, and then builds and validates the network model using the preprocessed images. The program is written in Python with the help of TensorFlow.

GitHub link: github.com/Mescetina/license-plate-recognition
